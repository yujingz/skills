# Personal Skills

This repository contains custom reusable skills for agent and workflow use.

## Structure
- `skills/`: Skill folders.
- `<skill>/SKILL.md`: Required file with trigger metadata and instructions.

## Included Skills
- `vps-3xui`: Generic Ubuntu VPS 3x-ui deployment and validation workflow covering DNS, reverse proxy, firewalling, protocol choice, and browser/CLI verification.
- `yujing-writing`: General writing-style skill for technical docs, client docs, and team docs (Chinese/English/bilingual).
