# Repository Instructions

## Skill Update Baseline

1. When creating or updating skills in this repository, keep output minimal by default.
2. Do not add a `references/` directory unless the user explicitly asks to add references.
3. Default skill shape should be `SKILL.md` only.
4. Add extra files or directories only when they are must-have for execution, such as `scripts/` or `assets/`.
5. If an optional resource is proposed, ask first unless the user has already requested it explicitly.
